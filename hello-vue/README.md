## 安裝一个第三方的插件

比如安装 markdown
```
## -S，–save 安装包信息将加到dependencies（生产阶段的依赖）

npm install --save @toast-ui/editor # Latest Version

## -D, –save-dev 安装包信息将加到devDependencies（开发阶段的依赖），所以开发阶段一般使用它

npm install --save-dev @toast-ui/editor # Latest Version
```
